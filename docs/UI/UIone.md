## UI简介

- （User Interface）：用户界面
- （User Interactiv）：用户交互

UI交互、UI界面、UI图标

## 风格

毛玻璃、扁平化、冷酷、轻拟物

| 表现层 | 视觉设计 |
| ------ | -------- |
| 框架层 | 设计界面 |
| 结构层 | 界面设计 |
| 范围层 | 交互设计 |
| 战略层 | 功能规格 |

动效、按钮、图片、配色、裁切

## 作业

摆放四个黑色正方形（可放大缩小旋转交叠）

表达出：顺序、递增、充满、醒目、紧张、嬉戏

提交三种方案
